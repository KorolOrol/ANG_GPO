# DataBase
---
## What is this for?
This part of the program is used to implement an embedded graph database, which will store plot data.
There can be a lot of DBs created and each one will store individual plot.

---
## What is using?

Base part and DB itself -- SliccDB. It's lightweight and EMBEDDED graph db fully written on C#.

In future it can be changed by self-written code (with same naming).
