# Cassius - нарративный генератор

Cassius - автоматизированная система для генерации элементов связного повествования по специальной нарративной модели, призванная сократить времязатраты сценаристов, писателей, нарративных дизайнеров, разработчиков игр на проработку сюжетных и литературных заготовок, дать референс и источник первичного вдохновения для дальнейшей качественной проработки. 

Репозиторий проекта ТУСУР ГПО ЭМИС-2401 **"Автоматическая генерация нарратива"**, итерация №1 (2024-2025).
